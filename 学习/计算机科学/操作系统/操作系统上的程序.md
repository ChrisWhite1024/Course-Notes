# 状态机与数字电路
## 数字逻辑电路
- 状态 = 寄存器保存的值
- 初始状态 = RESET
- 迁移 = 组合逻辑电路寄存器下一周期的值
# 什么是程序 (源代码视角)
程序就是状态机
##  C 程序的状态机模型
# 什么是（二进制）程序
- 状态 = 内存 + 寄存器

## 特殊的指令 syscall
- 把内存与寄存器状态完全交给 OS
- 与 OS 中其他对象交互
- OS 上的程序本质还是计算，但是还会调用 syscall 实现 IO

# 操作系统的一般程序
- 操作系统集中管理了所有的软硬件资源
- 在应用程序的视角上操作系统就是 syscall 的调用
- strace 可以追踪应用程序的系统调用
